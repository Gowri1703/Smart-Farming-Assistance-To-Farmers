# AgriSmart

**AgriSmart** is a farmer-focused web application that provides essential tools and information for farmers. It works both online and offline, making it easy to access weather updates, crop advisories, market prices, soil health info, pest detection, and more, even without internet.

---

## Features

- Set up and manage your **farmer profile**
- View **dashboard** with quick access to all features
- Check **weather updates** and **crop advisories**
- Monitor **soil health** and detect **pests**
- Access current **market prices**
- Chat with a **farming chatbot**
- Offline support with local caching
- Automatic data syncing when online
- Manage settings and logout
